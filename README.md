# WPR Portal – MVP (Vite + React + Tailwind)

## Lokalnie
```bash
npm install
npm run dev
```

## Deploy na Vercel
1. Wgraj repo na GitHub (zawartość tego ZIP-a).
2. Na Vercel: Add New → Project → Import Git Repository.
3. Framework: **Vite** (auto), Build: `npm run build`, Output: `dist` (domyślnie).
